# 🚀 유튜브 황금 주제 발굴기

데이터 기반의 11가지 지표를 분석하여, 조회수가 터질 확률이 높은 유튜브 주제를 발굴하는 대시보드입니다.

## 📋 프로젝트 개요

- **목표**: 조회수 성장 가능성이 높은 유튜브 주제 발굴
- **기술 스택**: Python, Streamlit, YouTube Data API v3
- **핵심 철학**: "숫자보다 중요한 것은 실제 영상이다."

## 🛠️ 설치 및 실행

### 1. 필요한 패키지 설치

```bash
pip install -r requirements.txt
```

### 2. 앱 실행

```bash
streamlit run main.py
```

### 3. API 키 설정

1. [Google Cloud Console](https://console.cloud.google.com/) 접속
2. 프로젝트 생성 또는 선택
3. YouTube Data API v3 활성화
4. API 키 생성
5. 앱 사이드바에 API 키 입력
