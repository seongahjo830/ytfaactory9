깃허브에 업로드하는 방법 (수동)

1. 터미널(명령 프롬프트 또는 PowerShell)을 열고
   현재 폴더로 이동하세요.

2. 다음 명령어를 순서대로 실행하세요:

   git init
   git remote add origin https://github.com/seongahjo830/searching.git
   git add .
   git commit -m "유튜브 황금 주제 발굴기 - 트렌딩 주제 발굴 기능 추가"
   git branch -M main
   git push -u origin main

3. GitHub 인증이 필요할 수 있습니다:
   - 사용자명: seongahjo830
   - 비밀번호: GitHub Personal Access Token 사용
     (GitHub > Settings > Developer settings > Personal access tokens)

또는 "깃허브업로드.bat" 파일을 더블클릭하세요!
