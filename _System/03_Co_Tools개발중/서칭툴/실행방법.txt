🚀 유튜브 황금 주제 발굴기 실행 방법

1️⃣ 필요한 패키지 설치
   터미널(명령 프롬프트 또는 PowerShell)에서 실행:
   
   pip install -r requirements.txt

2️⃣ 앱 실행
   터미널에서 실행:
   
   streamlit run main.py
   
   또는
   
   python -m streamlit run main.py

3️⃣ 브라우저에서 확인
   실행하면 자동으로 브라우저가 열립니다.
   주소: http://localhost:8501
