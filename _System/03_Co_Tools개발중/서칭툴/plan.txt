📄 [plan.md] 유튜브 황금 주제 발굴기 제작 계획서 🚀
1. 프로젝트 개요 🌟
목표: 데이터 기반의 11가지 지표를 분석하여, 조회수가 터질 확률이 높은 유튜브 주제를 발굴하고 해당 영상을 즉시 확인할 수 있는 대시보드 제작.

핵심 기술: Python, Streamlit, YouTube Data API v3.

핵심 철학: "숫자보다 중요한 것은 실제 영상이다." 모든 랭킹은 유튜브 영상 임베드 형태로 제공함.

2. 데이터 분석 지표 (7가지 단일 지표) 📊
각 지표는 0~100점으로 정규화하여 계산합니다.

조회수 효율: 영상 조회수 / 채널 구독자 수 (작은 채널이 대박 난 주제 찾기) 🎯

급등 에너지: 최근 7일간의 조회수 상승 폭 및 검색량 추이 🔥

블루오션 지수: 키워드 검색량 / 최근 업로드 영상 수 (수요는 많으나 공급이 적은 곳) 🌊

글로벌 트렌드 전이: 해외 동일 키워드의 폭발적 성장이 한국으로 전이되는 정도 🌍

콘텐츠 노후도: 상위 노출 영상들의 평균 제작 시기 (오래된 영상이 많을수록 기회) 🕰️

참여 밀도: (좋아요 + 댓글) / 조회수 (시청자 반응이 뜨거운 주제) 💬

롱테일 확장성: 관련 연관 검색어 및 질문형 키워드의 풍부함 🔗

3. 복합 및 최종 지표 (4가지) 🍹
폭발 성장형 (칵테일 A): 급등 에너지 + 글로벌 트렌드 조합 🚀

저리스크 침투형 (칵테일 B): 블루오션 + 콘텐츠 노후도 조합 🏹

팬덤 형성형 (칵테일 C): 조회수 효율 + 참여 밀도 조합 🤝

최종 마스터 추천: 위 7가지 지표에 가중치를 부여한 종합 점수 🏆

4. UI/UX 구현 계획 💻
사이드바: 11가지 랭킹 선택 메뉴, 검색어 입력, 가중치 조절 슬라이더.

메인 화면: 랭킹별 영상 갤러리 (2~3열 그리드 레이아웃).

영상 카드:

순위 배지 (예: 1st, 2nd...)

유튜브 임베드 플레이어 (st.video)

추천 이유 및 핵심 지표 요약 텍스트.

5. 단계별 개발 체크리스트 (Cursor 전용) ✅
이 체크리스트를 순서대로 완료하며 진행할 것.

[ ] Step 1: Streamlit 기본 구조 설정 및 Google API 키 연동 테스트.

[ ] Step 2: 특정 키워드에 대한 유튜브 검색 결과 및 상세 데이터 수집 함수 구현.

[ ] Step 3: 7가지 단일 지표 계산 알고리즘 개발 및 데이터프레임 생성.

[ ] Step 4: 3가지 칵테일 지표 및 최종 가중치 점수 합산 로직 구현.

[ ] Step 5: 메인 UI 구현 (영상 임베드 및 그리드 레이아웃).
