# 🚀 실행 방법

## 가장 간단한 방법

### 방법 1: 더블클릭으로 실행 (추천)
1. **`실행.bat`** 파일을 더블클릭
2. 자동으로 브라우저가 열립니다!

### 방법 2: 처음 설치할 때
1. **`설치및실행.bat`** 파일을 더블클릭
2. 패키지 설치 후 자동으로 실행됩니다

---

## 수동 실행 방법

### 1. 패키지 설치
```bash
pip install -r requirements.txt
```

### 2. 앱 실행
```bash
streamlit run main.py
```

---

## 사용 방법

1. **API 키 입력**
   - 사이드바에 YouTube Data API v3 키 입력
   - `KeyKey1.txt` 파일에 있는 키 사용 가능

2. **검색**
   - 검색 키워드 입력
   - "검색 실행" 버튼 클릭

3. **결과 확인**
   - 3열 그리드로 영상 표시
   - 랭킹 선택 및 가중치 조절 가능

---

## 문제 해결

- **브라우저가 안 열리면**: `http://localhost:8501` 직접 입력
- **오류가 나면**: `설치및실행.bat` 파일 실행 (패키지 재설치)
- **종료하려면**: 실행 창을 닫거나 `Ctrl + C`
