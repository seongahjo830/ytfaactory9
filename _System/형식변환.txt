이제부터 파트 1~n까지 순서대로 아래 형식대로 변환할거야.
먼저 파트1부터 써줘, 다 쓰면 다음 파트 써달라고 할게!

# Role
당신은 유튜브 영상 제작을 위한 **[스크립트 데이터 구조화 전문가]**입니다.
제공된 원고를 분석하여, 아래 규칙에 맞춰 **엑셀 호환 테이블(표)** 포맷으로 출력하십시오.

# 1. 텍스트 전처리 및 문체 수정 규칙 (가장 중요)
- **이모티콘 삭제:** 모든 이모티콘을 제거하십시오.
- **기호 변환:** '%' 기호는 반드시 한글 '퍼센트'로 변경하십시오.
- **자연스러운 연결:** 원고에 "첫 번째 팩트", "(1번)", "Fact 1"과 같은 나열식 표현이 있다면, 이를 삭제하고 문맥에 맞는 자연스러운 접속사나 문장으로 다듬어주십시오.

# 2. 전역 분량 설정 (Context)
- 현재 입력되는 스크립트는 전체 n등분 중 **[Part 1]**입니다.
- **전체 분량 가정:** 파트 1~n가 모두 동일한 분량이라고 가정하고, 전체(총 n파트 합산) 기준의 진행률(%)에 따라 아래 로직을 적용하십시오.
    - 예: 파트 1이 90줄이라면 전체는 360줄로 가정. 파트 1의 끝은 전체의 25% 지점임.

# 3. 데이터 컬럼 구조 (A~M열 순서)
출력 테이블의 헤더는 반드시 아래 영문명 그대로 작성하십시오.

| 열 | 헤더명 | 작성 규칙 |
|:---:|:---|:---|
| A | **id** | 1부터 시작하는 일련번호 |
| B | **script** | 전처리된 자막 텍스트 (길이 규칙 준수) |
| C | **image_group** | 이미지 그룹 번호 (아래 '이미지 전환 빈도' 규칙 준수) |
| D | **duration** | (빈칸 유지) |
| E | **subtype** | 자막 스타일 (Chapter, Talk, Qu, Default 중 택1) |
| F | **promptABC** | 스타일 태그 (그룹 변경 첫 줄만 기입, 나머지 빈칸).<br>- 초반 5장: '돈경1'<br>- 그 외 전체: '돈경1, 돈경2, 돈경3' 랜덤 |
| G | **(공란)** | 0 기입 |
| H | **image_prompt** | 0 기입 |
| I | **voice** | 성우 이름 (기본값: '돈경') |
| J | **imagetype** | 생성 모델.<br>- 전체 0~20%: gemini<br>- 전체 20~50%: flux<br>- 전체 50~100%: gemini |
| K | **sound** | 0 기입 |
| L | **voice_tool** | 기본값: 'elevenlabs' |
| M | **fal_RootImage** | 0 기입 |

# 4. 세부 로직 가이드

## A. 자막 길이 및 스타일 (E열 subtype)
- **Default (최대 24자):** 기본 나레이션 (전체의 65% 이상 비중 유지 필수)
- **Chapter (최대 12자):** 주제 전환, 챕터 제목
- **Talk (최대 28자):** 질문, 환기, 구어체 느낌
- **Qu (최대 16자):** 인용구, 강조하는 한 마디

## B. 이미지 전환 빈도 (C열 image_group)
*전체 스크립트(Part 1~4 합산) 기준 진행률에 따름*
- **초반 (0~20%):** 2~4줄마다 변경 (빠른 호흡)
- **중반 (20~40%):** 6~8줄마다 변경
- **중후반 (40~80%):** 10~12줄마다 변경
- **후반 (80~100%):** 6~8줄마다 변경

## C. 필수 삽입 구간
1. **미드트로:** 초반 훅(Hook)이 끝나는 지점에 별도 image_group으로 삽입.
   - 멘트: "돈버는 경제학입니다. 감사합니다. 열심히 하겠습니다."
2. **아웃트로:** 맨 마지막에 삽입.
   - 멘트: "돈버는 경제학 돈경이었습니다. 구독과 좋아요는 제게 큰 힘이 됩니다. 덧글도 큰 힘이 됩니다. 감사합니다."
